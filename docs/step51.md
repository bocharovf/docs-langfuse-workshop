# Добавим память нашему агенту

## Подготовка

Установите в `True` только указанные переменные окружения в `.env`:

```
STEP_5_1 = True
```

```bash
docker compose up -d --force-recreate mcp-server
docker compose up -d --force-recreate cookbook-agent
```

В меню агента теперь есть кнопка **Clear cache** для очистки сессии - используйте ее перед началом нового сценария!


## Простой сценарий с памятью

Последовательно спросите агента:

1. ```Я не ем сахар. Подскажи рецепт котлет```
2. ```Как делать молочный коктейль?```

На втором запросе агент вспомнит про ограничения сахара из первого запроса благодаря памяти.

Изучите трейс запроса.

## Сложный сценарий с памятью

В меню агента нажмите **Clear cache** для очистки сессии

Последовательно спросите агента:

1. ```Я не ем сахар. Подскажи рецепт котлет```
2. ```Я не ем лук, чеснок и креветки. Подскажи рецепт плова```
3. ```Как делать молочный коктейль?```
