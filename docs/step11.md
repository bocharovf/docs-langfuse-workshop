## Подготовка

Установите в `True` только указанные переменные окружения в `.env`:
```
STEP_1 = True
```
Остальные переменные должны быть равны `False`!

Перезапустите агент:
```
docker compose up -d --force-recreate cookbook-agent
```

Сделайте несколько запросов к агенту.
